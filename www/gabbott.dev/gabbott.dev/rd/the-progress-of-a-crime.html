The Progress of a Crime Review

This is my first real venture into crime stories of this kind, which I decided to pick up at Waterstones a year or two ago. The only other crime book I've ever really read apart from this is Nisioisin's Decapitation, which was very good, so I was hoping this was going to be as good.

Sadly it wasn't.

I don't have a massive amount to say about this as a book though, as it was just uninteresting. Not so bad it's worth talking about, just dull.

I might try and expand with some detail at some point, but that is about all I've gotten from this read. I bought some others at the same time, e.g. Fell Murder, Excellent Intentions, The Murder of my Aunt and Deep Waters, all a part of the same set of murder mystery books which are British Library Classics. I hope when I get around to reading these they are a bit more interesting.
