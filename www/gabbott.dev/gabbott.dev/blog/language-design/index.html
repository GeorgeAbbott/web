Language Design Series

The goal of this language will be the following:

  • A Ruby-like syntax, with, e.g. begin and end
  • Compiles to native code
  • Sane defaults: const-by-default, private-by-default, (pure-by-default?)
  • Easy to parse and simple: no extraneous features
  • Strong functional programming, like Rust, etc.
  • Still undecided on memory safety, what to do.
