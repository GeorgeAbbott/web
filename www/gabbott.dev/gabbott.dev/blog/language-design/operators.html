Operators

Thanks to the function oriented design, a lot of operators can actually just be functions instead. We will list them here with the operators though.

Operator Usage
+, -, /, *, %, %%, >>, <<, bitand, bitor, bitxor Actually defined as infix functions, e.g. add aka `+`; mod aka `%`, etc.
=, +=, -=, /=, *=, %=, %%= Assignment operators. = is an actual operator, the rest are functions defined in terms of that operator.
==, !=, <=, >=, <, > As functions, e.g. eq.
