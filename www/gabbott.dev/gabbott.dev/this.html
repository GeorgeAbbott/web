This Website

This page details the way that this website is set up, and how it changes over time.

Present Day

I'm planning on changing this, so watch this space, but currently, I use a Git repo. I make changes locally, hand-editting the HTML, comitting it to the Git repo, and pushing it. I have a webhook set up, so that when a new commit is pushed the repo, it is automatically pulled into a local folder on the server. Then, symlinks are set up the nginx and www folders, which host the webpage.

I'm planning to move from Git to rsync to sync my website.
