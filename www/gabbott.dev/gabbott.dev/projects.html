Projects

Here are some of projects that I work on in my spare time - both programming related, and non-programming.

Ongoing Projects

  • This website - specifically, updating the blog and writing scripts to update other parts of it.
  • The allotment - I want to get much better at it, and make it look very good in the coming year. See Allotment 2024 for some of my future plans.
  • orgd - This contains a collection of scripts which are used on my system. Some of them frankly don't belong, but are stuck here for now. A mix of Shell and Ruby scripts.
  • panic - A CLI for Stack Overflow. Nowhere near complete yet. I have a domain for when I complete it, at panicking.dev.
  • Skirt-lang - Attempting designing a systems programming language, but no implementation yet, and still deciding how I want it to look.
  • Dlang Game: writing a game in D, using SDL, to try out the language, and gamedev again.

Completed

  • EPQ - My EPQ I wrote for A-level. The topic was, How does language affect the way in which we think? (so, Sapir-Whorf essentially)
